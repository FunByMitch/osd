Quick Start

Get Oriented in Three Steps
1
Choose a pattern
Click the Gallery dropdown at the top of the left panel. Hover over any name to see a preview appear to the right. Click to select it.
2
Adjust parameters
Scroll down the left panel to find all controls — cell size, colors, dot density, arc depth, and display options. The canvas on the right updates instantly as you adjust.
3
Open or print
Scroll to the bottom of the panel. Open in new window renders a full 1920×1080 PNG. Print / save PDF renders at US Letter and opens your browser's print dialog.
Tip: Your last-used settings for each pattern are automatically saved. When you return to a pattern, it picks up exactly where you left off.

Geometric Patterns

Offset Dots
The foundation of the family. Diamond-shaped fills bounded by straight lines, arranged in an offset grid. Clean and geometric — each alternating strip gets Color 1 or Color 2.
Best for: Clean two-color patterns, quilting references, simple textile designs. Start here to understand the grid before exploring the arc variants.
Triangle Fill
Straight-line triangles derived from the same offset grid. Two triangle orientations (up and down) each get their own color, and the narrow edge triangles at the left and right boundaries get a third accent color.
Three colors: Unlike other patterns this one uses three distinct fills — use the Edge Triangle Color to either match a main color (to hide them) or contrast strongly to frame the pattern.
Sine Offsets
Adjacent boundaries are cosine waves in opposite phase, so they cross at every grid point and create interlocking lens or eye shapes. The boundary is a smooth continuous curve rather than piecewise arcs.
Cell dimensions matter: W controls the lens width and H controls the lens height. Square cells give round lenses; tall cells give narrow pointed lenses.
Hex Fill ×2
A honeycomb tiling derived from the offset dot grid with a two-color alternation. Each dot becomes the center of a regular hexagon. Only complete hexagons are drawn — no partial shapes at the edges.
Two-color note: Adjacent hexes in the same row always alternate colors, but some diagonal neighbors may share a color. This creates a striped zigzag rhythm across the pattern.
Hex Fill ×3
The same honeycomb tiling with a mathematically proper 3-coloring — no two adjacent hexes ever share a color. Uses axial coordinates to compute (q + 2r) mod 3 for each cell.
True 3-coloring: A hexagonal lattice requires exactly 3 colors for proper coloring. Every hex's 6 neighbors receive all three color values — guaranteed by the axial formula.

Arc Patterns

Curved Offsets
Arc boundaries alternate direction on every row — one segment bows left, the next bows right. Both edges of each strip mirror each other, creating a symmetric zigzag scallop effect.
Arc Depth effect: At low values (5–10) the pattern has a gentle wave. At high values (20–30) it becomes dramatic and interlocking.
Scallop — Bead Chain
Each column boundary maintains one constant bow direction along its entire length. Even columns bow one way, odd columns bow the other. Strips alternate between pinched (hourglass) and barreled (sausage) shapes.
Distinctive character: The only pattern where adjacent strips feel structurally different from each other. Try contrasting colors to emphasize the pinch/barrel rhythm.
Scallop — Fish Scale
All arc boundaries bow in the same direction with no alternation. Every strip curves the same way, creating a cascading fish-scale or overlapping roof-tile effect that flows in one direction.
Direction feel: The uniform bow creates a strong sense of directional flow. Try Horizontal axis for a different movement direction.
Scallop — Pagoda
Arc boundaries alternate direction band by band — every column bows the same way within a horizontal band, then the whole pattern reverses in the next band. Strip edges connect with straight lines, creating a stacked crescent or tiered pagoda silhouette.
Unique feature: The straight horizontal connectors are intentional — they define the pagoda tiers and distinguish this from Curved Offsets.

Parameters

Every Control Explained
Grid Shape
Cell Width
Horizontal spacing between grid points
Sets how wide each diamond or triangle cell is in pixels. Smaller values pack more cells across the screen. Larger values make fewer, bigger shapes. Range 20–100px.
Cell Height
Vertical spacing between grid points
Sets the height of each cell. Matching Width and Height creates square diamond proportions. Making Height larger than Width stretches diamonds vertically into elongated shapes. Range 20–100px.
Dot Size
Center dot density — 0 means no dots
Places a filled circle at the center of each fill region. 0 = no dots. Higher values make smaller dots relative to the cell. At 1 the dot nearly fills the region; at 6 it's a small accent mark. The dot color is set separately below.
Triangle Fill uses the triangle's inradius (largest inscribed circle) as the dot size reference rather than the diamond formula.
Arc Depth arc patterns only
How much arc boundaries curve
Controls the perpendicular displacement of each arc boundary from the straight line between two grid points. 0 = perfectly straight lines (same as Offset Dots). Higher values produce stronger, more dramatic curves.
A good starting range is 8–15 for subtle curves. Above 25 arcs become bold and may overlap adjacent strips at small cell sizes.
Layout
Padding
Empty border around the pattern
Sets a pixel gap between the canvas edge and where the pattern begins. 0 = pattern fills edge to edge. For screen use, 0–20 is typical. When printing, a larger padding acts as a margin on the page.
Axis
Vertical or Horizontal fill orientation
Vertical — fills run in columns top to bottom. Horizontal — fills run in rows left to right. This is a 90° rotation of the same pattern; all bow rules rotate with it.
Colors
Color 1
Fill for even strips (1st, 3rd, 5th…)
The primary fill color. In Vertical axis, this fills the 1st column strip from the left. Use the color picker or type a hex value.
Color 2
Fill for odd strips (2nd, 4th, 6th…)
The alternating fill color. Contrasting colors emphasize the pattern structure; similar colors create a more unified, tonal effect.
Color 3
C3 — role depends on pattern
C3 serves a different purpose depending on which pattern is selected:
Arc patterns & Sine Offsets — center dot fill color. Only visible when Dot Size is greater than 0. White on dark fills creates a bead effect; matching Color 1 or 2 creates subtle embossed dots.
Triangle Fill — edge triangle color. Fills the narrow triangles at the left and right borders (NS) or top and bottom borders (EW). Match C1 or C2 to hide them; use strong contrast to frame the pattern. Triangle Fill uses a separate Dot Color (Cdot) for its center dots.
Hex Fill ×3 — third hex fill color. Used alongside C1 and C2 so no two adjacent hexagons share a color. Hex Fill ×3 also has a separate Dot Color (Cdot) — giving it four color parameters in total: three fills plus one dot color.
Triangle Fill — additional colors
Down / Right color
Downward-pointing (Vertical) or rightward-pointing (Horizontal) triangles
The majority fill triangle type. In Vertical axis these point downward (▽).
Up / Left color
Upward-pointing (Vertical) or leftward-pointing (Horizontal) triangles
The alternating fill triangle type. In Vertical axis these point upward (△).
Edge color
Narrow triangles at the left and right edges
The offset grid leaves narrow triangular gaps at the left and right borders. These get their own color. Match Color 1 or 2 to hide them. Use a strong contrast to create a decorative frame.
Show Grid Dots
Small markers at every grid intersection
Draws 3.5px circles at each offset grid point. Useful for checking cell proportions or for a design aesthetic where the underlying grid is visible.
Hex Fill ×2 and ×3 — parameters
Cell size
Horizontal distance between hex centers (W)
Controls the size of each hexagon. H is derived automatically as W × √3 to keep hexagons regular. Only complete hexagons are drawn — partial hexagons at the edges are always skipped.
C1 · C2 / C1 · C2 · C3
Hex fill colors
Hex Fill ×2 alternates two colors using a row + column offset. Some diagonal neighbors may share a color, creating a zigzag stripe rhythm.

Hex Fill ×3 uses a mathematically proper 3-coloring via axial coordinates — no two adjacent hexagons ever share a color. Requires three distinct fill colors.
Dot color
Center dot fill — fourth color in Hex Fill ×3
Hex Fill ×3 has four color parameters total: C1, C2, C3 for hex fills plus a separate Dot Color (Cdot). Set Dot Size to 0 to hide dots entirely.
Show edges
Thin stroke on hex cell boundaries
Draws a faint outline around each hexagon, revealing the underlying honeycomb structure. Turn off for clean filled output.
Display Overlays arc patterns only
Show Lines
Overlay arc boundaries as thin black lines
Draws each arc as a thin black line on top of the fills. Useful for understanding the structure of a pattern, checking bow alignment, or as a deliberate design element. Turn off for clean final output.
Show Grid Points
Reveal the underlying offset dot grid
Shows the grid of offset dots that the pattern is constructed from — small black circles at every grid intersection. Each row is offset by half a cell width from the row above. Useful for understanding how the diamond grid works.

Output

Five Ways to Get Your Design Out
↗
Open in new window
Renders the pattern at 1920 × 1080px and opens it as a PNG image in a new browser tab. Right-click the image and choose Save image as… to download it. The pattern fills the full canvas with the same parameters you see in the preview — same W, H, colors, and padding.
⎙
Print / save PDF
Renders the pattern at US Letter size (8.5" × 11") and opens your browser's print dialog. Choose Save as PDF to get a print-ready file, or send directly to a printer. The pattern uses the padding you set as the page margin.
↓
Export preset
Downloads all current parameter values for the selected pattern as a JSON file (e.g. scallop-1-preset.json). Use this to save a combination you want to return to, share settings with someone else, or build a personal library of color palettes.
↑
Import preset
Loads parameters from a previously exported JSON file. The pattern will switch to match the preset immediately. Any parameter not present in the file falls back to the pattern's default value.
↺
Reset to defaults
Restores all parameters for the current pattern to their original factory values. Your changes are cleared and the last-saved state is overwritten. This cannot be undone — export a preset first if you want to keep your current settings.

Workflow Tips

Getting the Most out of the Studio

Start with Cell Size

Cell Width and Height define everything else. Get the grid density right before touching colors or arc depth. A good starting density for screen viewing is W=54, H=54. For print at US Letter, values around W=40–60 give 10–15 columns — enough to show the pattern character clearly.

Color Contrast Drives Impact

High contrast between Color 1 and Color 2 emphasizes the strip structure. Low contrast creates a more textile-like, tonal result. Bead Chain (Scallop 1) benefits most from contrast because the pinch/barrel alternation becomes obvious. Fish Scale (Scallop 2) works well with analogous colors — the directional flow reads clearly even with similar hues.

Arc Depth Sweet Spots by Pattern

Curved Offsets: 8–12 for gentle; 18–25 for dramatic. Bead Chain: 10–15 — beyond 20 the hourglass pinch can close off entirely. Fish Scale: 10–18. Pagoda: 8–14 — the straight horizontal borders mean arc depth only affects the vertical columns, so moderate values work best.

Axis as a Design Tool

Don't just use Vertical by default. Horizontal axis gives Fish Scale a completely different feel — it reads as overlapping roof tiles rather than fish scales. For Pagoda, Horizontal axis creates stacked columns rather than rows, which has a different architectural quality.

Use Show Lines to Understand a Pattern

If a pattern's structure isn't clear, turn on Show Lines. The thin black arc overlay reveals exactly how the boundaries are constructed. Turn it off for final output.

Save Presets as You Explore

Whenever you find a color combination you like, export a preset before changing anything. The studio saves your last-used state automatically, but export gives you a permanent named file you can return to any time.

Printing tip: Set Padding to 40–54px before printing — this gives a comfortable white margin on the page. The pattern won't change visually on screen, but the printed output will have clean borders.
